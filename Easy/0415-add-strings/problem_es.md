# 415. Sumar Cadenas

[Enlace a LeetCode](https://leetcode.com/problems/add-strings/)

## Descripción

Dados dos enteros no negativos, `num1` y `num2` representados como cadenas, retorna *la suma de* `num1` *y* `num2` *como una cadena*.

Debes resolver el problema sin usar ninguna librería integrada para manejar enteros grandes (como `BigInteger`). Tampoco debes convertir las entradas a enteros directamente.

## Ejemplos

**Ejemplo 1:**

```text
Entrada: num1 = "11", num2 = "123"
Salida: "134"
```

**Ejemplo 2:**

```text
Entrada: num1 = "456", num2 = "77"
Salida: "533"
```

**Ejemplo 3:**

```text
Entrada: num1 = "0", num2 = "0"
Salida: "0"
```

## Restricciones

* `1 <= num1.length, num2.length <= 10^4`
* `num1` y `num2` consisten solo de dígitos.
* `num1` y `num2` no tienen ceros a la izquierda excepto por el cero mismo.
